#### Bookkeeper is a comprehensive digital library management system for book enthusiasts.

## Features

-   Track your reading progress across multiple books
-   Organize collections by genre, author, or custom tags
-   Advanced search functionality
-   Reading statistics and insights
-   Integration with popular book APIs

## Technology

Built with React, Node.js, and MongoDB. Utilizes Google Books API for metadata.

[GitHub Repository](https://github.com/yay5a/book-search)

_Note: This project is currently under active development and not yet deployed to production._
