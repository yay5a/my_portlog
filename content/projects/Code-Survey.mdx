# Developer Survey Platform

A data collection platform designed to gather meaningful insights about developers' learning paths and professional journeys.

## Features

-   Anonymous submission of coding experiences
-   Interactive form with branching logic
-   Data visualization dashboard
-   Mobile-responsive design

## Technology

Built with HTML, CSS, JavaScript, and leverages Google Forms API for data collection.
